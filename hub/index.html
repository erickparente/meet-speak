Como adicionar um novo curso ou aplicativo
  1. Crie e publique o app onde preferir (Lovable, Claude, GitHub Pages…) e copie a URL publicada.
  2. No GitHub, abra o arquivo hub/apps.json deste repositório e clique no lápis (editar).
  3. Copie um dos blocos existentes da lista "apps", cole no fim da lista e ajuste: nome, descricao, emoji, categoria, a url publicada e "status": "ativo".
  4. Salve (Commit changes). O novo app aparece aqui em instantes.
  1. Para tirar um app do ar sem apagar, troque o status para "em-breve".
  2. Para criar uma categoria nova, acrescente-a em "categorias" e use o mesmo id no app.
  3. Para renomear a plataforma, edite "plataforma" no mesmo arquivo.
